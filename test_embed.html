üíª Integration Code

This is the code used to embed the widget on this page. Copy it to use on your own website!

<!-- LifeLink Chatbot Widget -->
<script>
  window.LifeLinkConfig = {
    apiUrl: 'http://localhost:5000',
    primaryColor: '#ff4757',
    buttonColor: '#ffa7c4',
    textColor: '#ffffff',
    position: 'bottom-right',
    buttonSize: 56,
    windowWidth: 400,
    windowHeight: 600,
    fontFamily: 'Poppins, sans-serif',
    brandName: 'SkillFusion'
  };
</script>
<script src="http://localhost:5000/widget.js"></script>
üí° Pro Tip: For production, replace http://localhost:5000 with your actual domain!

‚öôÔ∏è Configuration Options

Available Settings:

  • apiUrl: Your API server URL (required)
  • primaryColor: Header and button colors (default: #ff4757)
  • buttonColor: Floating button color (default: #ffa7c4)
  • textColor: Text color on colored backgrounds (default: #ffffff)
  • position: 'bottom-right' or 'bottom-left' (default: bottom-right)
  • buttonSize: 40-80 pixels (default: 56)
  • windowWidth: 300-600 pixels (default: 400)
  • windowHeight: 400-800 pixels (default: 600)
  • fontFamily: Any Google Font (default: Poppins)
  • brandName: Your company name (default: SkillFusion)
