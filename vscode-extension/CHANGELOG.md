# Change Log

All notable changes to the "LifeLink's Buddy - Weather Companion" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [2.0.0] - 2025-11-06

### Added
- **Auto weather updates** - Get notifications 3 times daily (8 AM, 2 PM, 8 PM)
- **Smart weather reminders** - "Don't forget your umbrella!" on rainy days
- **Auto-location detection** - IP-based geolocation, no manual setup needed
- **Beautiful p5.js animations** - Floating particles and weather effects
- **Tabler Icons integration** - Professional icon system throughout UI
- **Manrope font** - Modern, clean typography
- **Weather photography** - Stunning images for rainy, snowy, and sunny conditions
- **Status bar widget** - See current weather at a glance
- **Explorer sidebar view** - Non-intrusive dropdown in Explorer panel

### Changed
- **Complete redesign** - Transformed from chatbot to weather companion
- **Moved to Explorer sidebar** - Removed Activity Bar icon for cleaner interface
- **Webpack bundling** - All dependencies now bundled for faster loading
- **Improved performance** - Optimized with webpack production builds
- **Enhanced UI** - Modern gradient design with animations
- **Better UX** - One-time setup with auto-location detection

### Fixed
- **Content Security Policy** - Fixed Tabler Icons font loading
- **Axios dependency bundling** - Solved "Cannot find module 'axios'" error
- **Asset path issues** - Corrected paths for weather images
- **Icon display** - Fixed Tabler Icons not showing due to CSP

### Technical
- **Webpack 5.102.1** - Complete bundling solution
- **TypeScript 5.0** - Type-safe development
- **p5.js 1.7.0** - Canvas animations
- **Axios 1.13.2** - HTTP client (bundled)
- **OpenWeatherMap API** - Real-time weather data
- **IP-API** - Auto-location detection
