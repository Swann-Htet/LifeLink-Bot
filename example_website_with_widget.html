üíª The Code Used on This Page

This is ALL the code needed to add the chatbot to this website: 

<!-- LifeLink Disaster Response Chatbot -->
<script>
  window.LifeLinkConfig = {
    // No apiUrl needed - automatically uses widget.js domain!
    primaryColor: '#ff4757',
    buttonColor: '#ffa7c4',
    position: 'bottom-right',
    buttonSize: 56,
    windowWidth: 400,
    windowHeight: 600,
    brandName: 'Example Company'       // ‚Üê Their company name
  };
</script>
<script src="http://your-domain.com/widget.js"></script>  ‚Üê Change to YOUR domain!
